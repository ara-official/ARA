# ARA
### Run [front-end] server (mac)
1. build
<pre><code>
$ cd ara/front-end/study/react_ara
$ ./0_install.sh
$ ./0_1_add_redux.sh
</code></pre>

2. run
<pre><code>
$ ./1_start.sh
</code></pre>

### Run [back-end] server (?)
1. django 및 필요 패키지 설치
<pre><code>
$ pip install django
$ pip install djangorestframework
$ pip install django-rest-swagger
</code></pre>
2. Run back-end
<pre><code>
$ python manage.py runserver [ip:port]
</code></pre>
